# Sarthak-s-vityarthi_aiml-project
“Smart Task Manager with Priority &amp; Analytics” A Python‑based command‑line task manager that:  Adds, edits, deletes, and marks tasks as done.  Uses data structures (lists/dicts + optional priority queue / heap).  Categorizes and filters tasks. Tracks basic analytics like:  Total tasks.  Completed vs pending.  Average completion time per category.
